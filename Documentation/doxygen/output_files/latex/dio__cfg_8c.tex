The following array contains the configuration data for each digital input/output peripheral channel (pin). Each row represent a single pin. Each column is representing a member of the \mbox{\hyperlink{struct_dio_config__t}{Dio\+Config\+\_\+t}} structure. This table is read in by Dio\+\_\+\+Init, where each channel is then set up based on this table.
